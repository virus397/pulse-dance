# Pulse Dance Assets

Drop your `.mp3` tracks into this folder. Ensure they match these exactly:

- `calm.mp3`: Base loop for the Calm mood.
- `anxious.mp3`: Base loop for the Anxious mood.
- `playful.mp3`: Base loop for the Playful mood.
- `encounter.mp3`: The overtone/layer that fades in when someone else connects.

> Note: All tracks should ideally be loopable stems or ambient soundscapes that seamlessly repeat, as they will loop infinitely once the app connects.
